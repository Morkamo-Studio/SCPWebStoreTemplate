# SCP:SL Web Store Template

Учебный шаблон веб-магазина внутриигровых привилегий, созданный для практики HTML, CSS и JavaScript.

Репозиторий сохранён как архив одной из моих ранних работ. Проект не является официальным сайтом SCP Foundation, Northwood Studios или SCP: Secret Laboratory и не связан с их разработчиками.

Кнопки покупки и авторизации являются элементами интерфейса и не подключены к реальной платёжной или серверной системе.

## Технологии

- HTML
- CSS
- JavaScript

## Графика

Вся графика в проекте создана мной, кроме логотипа SCP Foundation, использованного как элемент фона.

Автор оригинальной графики и дизайна: **Kirill Drozdov / Mercury Morkamo**.

Графику проекта можно использовать и изменять при условии указания автора и распространения производных материалов на условиях **CC BY-SA 3.0**.
